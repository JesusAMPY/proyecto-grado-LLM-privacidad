# Marco de Privacidad LLM para Educación Superior Colombiana

## 📋 Descripción del Proyecto

Este proyecto presenta un **Marco integral para la evaluación y gestión de riesgos de privacidad en la adopción de Modelos de Lenguaje Grande (LLM)** en Instituciones de Educación Superior colombianas. El marco está alineado con la **Ley 1581 de 2012** (Protección de Datos Personales), **CONPES 4144** (Estrategia Nacional de IA), **OWASP Top 10 para LLMs**, **Marco de Ciberseguridad NIST** e **ISO/IEC 27701**.

### 🎯 Objetivo Principal

Proporcionar a las IES colombianas un recurso educativo y práctico para:
- Identificar y evaluar riesgos de privacidad en la adopción de LLM
- Garantizar cumplimiento normativo con regulaciones colombianas
- Implementar controles y mitigaciones de riesgos efectivas
- Facilitar toma de decisiones informadas sobre adopción de IA

## 📱 Descripción del Prototipo

El prototipo es una **plataforma web educativa interactiva** que implementa el marco de privacidad LLM con las siguientes características:

### Componentes Principales

1. **Landing Page** - Introducción al marco con estadísticas y llamadas a la acción
2. **Descripción General del Marco** - Explicación de los 4 pilares fundamentales
3. **Catálogo de Riesgos** - 8+ riesgos de privacidad identificados con:
   - Clasificación por categoría (fuga de datos, comportamiento del modelo, acceso al sistema, normativo)
   - Niveles de severidad (crítico, alto, medio, bajo)
   - Ejemplos prácticos para contexto colombiano
   - Estrategias de mitigación
   - Impacto regulatorio

4. **Mapeo de Cumplimiento** - Alineación con:
   - Ley 1581 de 2012
   - CONPES 4144
   - OWASP Top 10 para LLMs
   - NIST Cybersecurity Framework
   - ISO/IEC 27701

5. **Casos de Estudio** - 4 escenarios reales de implementación en IES:
   - Asesoramiento académico con LLM
   - Investigación y análisis de datos
   - Recursos humanos e interacciones
   - Seguridad y análisis de amenazas

6. **Prototipo Interactivo** - Dashboards y visualizaciones:
   - Flujo de Evaluación de Riesgos
   - Cronograma de Implementación (6 meses)
   - Mapa de Calor de Severidad de Riesgos
   - Mapeo de Cumplimiento Regulatorio
   - Escenarios de Flujo Real

7. **Recursos & Referencias** - Guías, plantillas y bibliografía especializada

### Características Técnicas

- ✅ **Completamente en Español** - Terminología colombiana y regulatoria
- ✅ **Diseño Responsivo** - Funciona en desktop, tablet y móvil
- ✅ **Colores Vibrantes** - Rojo (#e74c3c), Azul (#3498db), Naranja (#f39c12)
- ✅ **Accesible** - WCAG 2.1 AA, navegación clara, alto contraste
- ✅ **Rendimiento** - Páginas pre-renderizadas estáticamente
- ✅ **SEO Optimizado** - Metadatos y estructura semántica

## 🎓 Conclusiones

### Hallazgos Principales

1. **Brecha Identificada:** Las IES colombianas carecen de guías estructuradas para adopción segura de LLM
2. **Marco Efectivo:** Los 4 pilares (Evaluación, Clasificación, Cumplimiento, Controles) proporcionan cobertura integral
3. **Alineación Regulatoria:** El marco mapea exitosamente a todas las regulaciones aplicables
4. **Aplicabilidad Real:** Los casos de estudio demuestran viabilidad en contextos institucionales reales

### Contribuciones al Conocimiento

- Primer marco específico para privacidad en LLM en contexto colombiano
- Adaptación de estándares internacionales a regulación local
- Prototipo funcional demostrable para adopción institucional
- Base para futuras investigaciones en IA responsable

### Recomendaciones de Implementación

1. **Corto Plazo (0-3 meses):** Validación con DPOs y responsables de TI de 5-10 IES piloto
2. **Mediano Plazo (3-6 meses):** Integración de módulo de autenticación y datos personalizados
3. **Largo Plazo (6-12 meses):** Certificación institucional del marco y formación de facilitadores

### Impacto Esperado

- Reducción de riesgos de privacidad en adopción de LLM
- Mayor cumplimiento con Ley 1581 en sector educativo
- Empoderamiento de IES para tomar decisiones informadas sobre IA
- Contribución a ecosistema de IA responsable en Colombia

## 🚀 Cómo Ejecutar el Proyecto

### Requisitos Previos
- Node.js 18+
- PNPM (o npm/yarn)

### Instalación

```bash
# Clonar el repositorio
git clone https://github.com/JesusAMPY/proyecto-grado-LLM-privacidad.git
cd proyecto-grado-LLM-privacidad

# Instalar dependencias
pnpm install

# Ejecutar servidor de desarrollo
pnpm dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

### Despliegue a Producción

```bash
# Construir para producción
pnpm build

# Iniciar servidor de producción
pnpm start
```

Desplegado automáticamente en Vercel con cada commit a `main`.
